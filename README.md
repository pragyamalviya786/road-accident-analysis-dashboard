# 🚦 Road Accident Analysis Dashboard

An interactive and data-driven Power BI dashboard developed to analyze road accident trends, casualties, and road safety patterns using visual analytics. This project transforms raw accident data into meaningful insights through KPI cards, trend analysis, maps, and comparative visualizations.

---

## 📊 Dashboard Features

- Total CY Casualties & Accident Analysis
- CY vs PY Casualties Trend Comparison
- Casualties by Vehicle Type
- Casualties by Road Type
- Urban vs Rural Area Analysis
- Light & Weather Condition Insights
- Interactive Accident Location Mapping
- Monthly Casualties Trend Visualization

---

## 🛠️ Tools & Technologies Used

- Power BI
- DAX
- Excel
- Data Visualization
- Business Intelligence

---

## 🎯 Project Objective

The objective of this dashboard is to identify accident patterns, casualty distribution, and high-risk road conditions through interactive visual storytelling. The dashboard helps in understanding accident trends and supports data-driven road safety analysis.
